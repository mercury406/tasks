# Tasks project

### Prerequisites
```
php 8.2.4
Laravel 10.48.18
```

### Run project

Start server at port 8000
```
php artisan serve --port=8000
```

See available routes
```
php artisan route:list
```

Send request using **HTTP client** (Postman, Insomnia, etc)

### Test

```
php artisan test
```
